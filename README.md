# RealtimeChart
Realtime chart code sample with Express Js, MongoDB, Socket IO, and ChartJS

## Install

```
npm install
```

## Usage

```
node index.js
```

It will show output :

```
Open http://localhost:3000
```

Then open http:://localhost:3000 on your browser.
